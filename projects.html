Project Description

Developed a machine learning model to predict the risk of heart disease in patients based on clinical features. The project involved the full ML pipeline — from raw data ingestion all the way to a deployed interactive web application.

Leveraged the Cleveland Heart Disease dataset with 13 clinical features including age, cholesterol, resting blood pressure, and ECG results. Applied multiple classification algorithms and selected the best-performing model achieving ~88% accuracy.

Key Highlights
Full ML pipeline: data ingestion → preprocessing → modeling → deployment
Feature engineering and selection from 13 clinical attributes
Data preprocessing: handling missing values, normalization, encoding
Evaluated multiple classifiers — Logistic Regression, Random Forest, SVM
Achieved ~88% accuracy on test set with cross-validation
Deployed as an interactive web app for real-time prediction

Project Description

Built a comprehensive data analysis pipeline using Python to explore and visualize UCL (UEFA Champions League) datasets. The goal was to uncover meaningful trends, patterns, and statistical insights from historical match and performance data.

The pipeline covered the full data lifecycle — ingestion, cleaning, transformation, exploratory analysis, and visualization — using industry-standard Python libraries. Findings were presented through rich, annotated charts and summary statistics.

End-to-end data pipeline from raw CSV ingestion to final visualizations
Data cleaning: handling nulls, duplicates, type casting, outlier detection
Exploratory Data Analysis (EDA) to surface trends and anomalies
Statistical summaries: distributions, correlations, group aggregations
Rich visualization suite — bar, line, heatmap, and scatter plots
Insights on team performance, goal trends, and match outcomes
